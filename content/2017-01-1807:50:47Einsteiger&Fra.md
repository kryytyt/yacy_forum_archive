das hängt ganz davon ab, welchen der Knöpfe Sie zum speichern der
Einstellungen verwendet haben.\
Ein Klick auf dauerhaft speichern, legt die Einstellungen im
Local-Storage Ihres Browsers ab (Hierfür muss JavaScipt aktiviert sein)
und ihr Browser muss so wie in der Standardeinstellung so konfiguriert
sein, dass er dies zulässt. Des Weiteren werden diese Einstellungen beim
Löschen der Browser-Daten entfernt, wenn z.B. beim Firefox dabei ein
Haken bei \"Offline Website Data\" gesetzt ist.\
\
Ein Klick auf \"Startseite für einmalige Nutzung generieren\" erstellt
einen Link, den Sie sich als Bookmark bzw. Startseite einstellen können,
sodass auf dieser die gewählten Einstellungen vorhanden sind.\

[YaCy]{.underline} muss einfach nur an [einer]{.underline} geeigneten
Stelle entpackt werden. Programme zum Entpacken für Microsoft Windows
sind z.B. \[http://www.filzip.com/ FilZip\], \[http://www.winzip.com/
WinZip\] oder \[http://www.winrar.de/ WinRAR\] und für Mac OS X
\[http://www.stuffit.com/mac/index.html StuffIt Expander\]. Unter Linux
kann man \[http://perso.wanadoo.fr/coquelle/karchiver/ kArchiver\]
benutzen oder einfach die Befehle \'gunzip \<release\>.tar.gz\' und
\'tar -xf \<release\>.tar\' (oder kurz \'tar -zxf \<release\>.tar.gz\')
in der Kommandozeile eingeben.

</div>

<div>

Sollte sich bereits eine ältere Installation auf ihrem System befinden,
[können]{.underline} Sie die Einstellungen und Daten dieser älteren
Installation in die neuere Installation übernehmen. Dies lässt sich
einfach bewerkstelligen, indem Sie das Verzeichnis DATA aus dem
Stammverzeichnis der älteren Installation in das Stammverzeichnis der
neueren Installation verschieben. \'\'\'Sie sollten den Ordner auf jeden
Fall verschieben und nicht kopieren, da dieser Ordner sehr viele Dateien
enthält und das Kopieren eine sehr lange Zeit in Anspruch nehmen
würde.\'\'\' Haben Sie die Daten einer älteren Installation übernommen,
so [ist keine weitere Konfiguration]{.underline} mehr
[notwendig]{.underline}.

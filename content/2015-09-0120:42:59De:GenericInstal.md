=== Entpacken von YaCy ===

</div>

−

<div>

Nach dem \[http://www.yacy.net/~~yacy/Download.html~~ Herunterladen\]
muss YaCy einfach nur an eine geeigneten Stelle entpackt werden.
Programme zum Entpacken für Microsoft Windows sind z.B.
\[http://www.filzip.com/ FilZip\], \[http://www.winzip.com/ WinZip\]
oder \[http://www.winrar.de/ WinRAR\] und für Mac OS X
\[http://www.stuffit.com/mac/index.html StuffIt Expander\]. Unter Linux
kann man \[http://perso.wanadoo.fr/coquelle/karchiver/ kArchiver\]
benutzen oder einfach die Befehle \'gunzip \<release\>.tar.gz\' und
\'tar -xf \<release\>.tar\' (oder kurz \'tar -zxf \<release\>.tar.gz\')
in der Kommandozeile eingeben.

</div>

die neuere Installation übernehmen. Dies lässt sich einfach
bewerkstelligen, indem Sie das Verzeichnis DATA aus dem Stammverzeichnis
der älteren Installation in das Stammverzeichnis der neueren
Installation verschieben. \'\'\'Sie sollten den Ordner auf jeden Fall

sind an dieser Stelle mit der Installation fertig.

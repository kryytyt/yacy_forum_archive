added a \'id\' search modifier which creates only a single search result.
=========================================================================

Date: 2015-04-30 18:26:40

``` {style="white-space:pre-wrap;width:81ex"}
added a 'id' search modifier which creates only a single search result.
This looks like "id:593808139012694017". This makes it possible to
search messages with a specific id. This shall be used to view single
message landing-pages.
```

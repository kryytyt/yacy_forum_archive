if we have a backend peer which we don\'t use to push, search it only if
========================================================================

Date: 2015-12-23 11:12:41

``` {style="white-space:pre-wrap;width:81ex"}
if we have a backend peer which we don't use to push, search it only if
the search result set in our own index is too small to fulfill the
required count of the search request. This shall be used to chain search
requests in large storage set-up.
```

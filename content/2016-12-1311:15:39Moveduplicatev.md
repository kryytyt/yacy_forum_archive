============================================

Date: 2016-12-13 11:15:39

``` {style="white-space:pre-wrap;width:81ex"}
Move duplicate variables into it's own file

`installation.sh` and `start.sh` has the same variables with the same
values for each variables.

To increase maintainability, it's a good idea to move the same code
that has the same procedures or instructions into it's own code.
```

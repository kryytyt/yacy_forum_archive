Hello, you can use Advanced Crawler page (/CrawlStartExpert.html),
select \"From File\" Start Point, and paste url of a file containing
your urls list (one url per line).\
Be aware whole file content will be loaded in memory, so you have to
check sufficient free memory is available for YaCy : check file size,
and check free memory in /PerformanceMemory\_p.html (\"Now before GC\"
column).

<div>

Im zweiten Schritt wird nun diese Liste in YaCy importiert. Wählen Sie
zunächst in der Navigationsleiste unter „ Production" den Eintrag
„Advanced Crawler" aus. Es wird die Seite „Expert Crawl Start"
angezeigt, in der Sie einen Crawl-Job anlegen können. Aktivieren Sie
unter „Start Point" die Option „From File (enter a path within youtr
local file system)" und geben Sie im Eingabefeld daneben den Pfad zu der
html-Datei mit den Bookmarks an.  Passen Sie ggf. weitere Parameter für
den Expert Crawl an und klicken Sie ganz unten auf die Schaltfläche
„Start New Crawl Job".

</div>

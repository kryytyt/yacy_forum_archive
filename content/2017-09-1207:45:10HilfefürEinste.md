Hallo Crystalgazer ,\
I would also just add (sorry also for the English), that in addition to
the seeds list overriding, in the Network Configuration
(/ConfigNetwork\_p.html page) you also have a feature that could fit
your needs : the \"Public cluster\" in the \"Robinson Mode\" section.
You can specify there a list of comma separated yacy nodes addresses
(ip1:port1, ip2:port2 \...) that will be then the only remote peers to
be requested when performing a search request. This feature will only
work in \"Community-based web search\" mode (see /ConfigBasic.html).\

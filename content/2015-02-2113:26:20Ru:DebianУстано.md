[== Настройка автоматического обновления ==]{.underline}

</div>

 

\+

<div>

[При настройке этим способом, внутреннее авто обновление YaCy - не
работает. Автоматическое обновление может быть выполнено средствами  ОС
Ubnutu. например с помощью команды \[\[crontab\]\]. Для этого необходимо
вставить следующую строку в файл: /etc/crontab .]{.underline}

</div>

 

\+

<div>

[0 6 \* \* \* root apt-get update && apt-get -y \--force-yes install
yacy]{.underline}

</div>

 

\+

<div>

[]{.underline}

</div>

 

\+

<div>

[В Ubuntu, в выше переведенной строке действительна только для
системного файла crontab (/etc/crontab) - вы можете редактировать этот
файл, на более новых ОС Ubuntu напрямую, без помощи команды
crontab.]{.underline}

</div>

 

\+

<div>

[]{.underline}

</div>

 

\+

<div>

[In Ubuntu, the above line is only valid for the system crontab file
(located at /etc/crontab) - you can edit  this file on newer Ubuntu OS

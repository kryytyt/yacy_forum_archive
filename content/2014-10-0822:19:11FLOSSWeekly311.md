**Guests:** [Andrei Alexandrescu](http://dlang.org/) and [Walter
Bright](http://dlang.org/).

D is a general purpose systems and applications programming language. It
is a high level language, but retains the ability to write high
performance code and interface directly with the operating system API\'s
and with hardware (dlang.org).

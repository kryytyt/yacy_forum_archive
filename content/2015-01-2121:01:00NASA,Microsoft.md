NASA, Microsoft Collaboration Will Allow Scientists to \'Work on Mars\'
=======================================================================

Date: 2015-01-21 21:01:00

[![A screen view from OnSight, a software tool developed by JPL in
collaboration with
Microsoft](http://www.jpl.nasa.gov/images/mars/20150121/mars20150121-226.jpg){width="100"
height="75"}](http://www.jpl.nasa.gov/news/news.php?release=2015-025&rn=news.xml&rst=4451)\
\
NASA and Microsoft have teamed up to develop software called OnSight, a
new technology that will enable scientists to work virtually on Mars
using wearable technology called Microsoft HoloLens.

\* Java - YaCy benötigt eine vorinstallierte Java Laufzeitumgebung (JRE)
bzw. Java Entwicklungsumgebung (JDK), Version 7.0 oder höher ist
erforderlich.  Sollte Java auf Ihrem System noch nicht vorhanden sein,
so müssen Sie es vor der Installation von YaCy installieren. Java ist
z.B. von \[http://openjdk.java.net/install/ OpenJDK\] oder auf der
\[http://java.com/de/download/index.jsp Internetseite der Firma Oracle\]
erhältlich. Möchten Sie lediglich Java-Programme laufen lassen,
benötigen Sie \[http://de.wikipedia.org/wiki/JRE JRE\], die Java Runtime
Environment (zu deutsch: Java Laufzeitumgebung). Möchten Sie Programme
in Java entwickeln und vielleicht sogar an der Weiterentwicklung von
YaCy teilnehmen, so benötigen Sie das \[http://de.wikipedia.org/wiki/JDK
JDK\], das Java Development Kit (zu deutsch: Java Entwicklungsumgebung).

</div>

<div>

[===Entpacken von YaCy===]{.underline}

</div>

 

\+

<div>

[YaCy muss einfach nur an einer geeigneten Stelle entpackt werden.
Programme zum Entpacken für Microsoft Windows sind z.B. FilZip, WinZip
oder WinRAR und für Mac OS X StuffIt Expander. Unter Linux kann man
einfach den Befehl \'tar -zxf \<release\>.tar.gz\' in die Kommandozeile
eingeben.\<br /\>\<br /\>]{.underline}

</div>

 

\+

<div>

 

</div>

 

\+

<div>

[===Bestehende Konfiguration und Daten übernehmen===]{.underline}

</div>

 

\+

<div>

[Sollten Sie bereits über eine ältere Installation verfügen, können Sie
die Einstellungen und Daten dieser älteren Installation in die neuere
Installation übernehmen. Dies lässt sich einfach bewerkstelligen, indem
Sie das Verzeichnis DATA aus dem Stammverzeichnis der älteren
Installation in das Stammverzeichnis der neueren Installation
übernehmen. Sie sollten das DATA-Verzeichnis nach Möglichkeit
verschieben und nicht kopieren, da dieser Ordner sehr viele Dateien
enthält und das Kopieren sehr lange Zeit in Anspruch nehmen kann. Haben
Sie die Daten einer älteren Installation übernommen, so ist keine
weitere Konfiguration mehr notwendig]{.underline}. \<br/\>

</div>

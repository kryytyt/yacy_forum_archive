Java peut être téléchargé et utilisé gratuitement. S\'il n\'y a pas
[encore]{.underline}
d\'\[http://fr.wikipedia.org/wiki/Environnement\_d\'exécution\_Java
environnement Java\] sur votre système, vous devez l\'installer avant
d\'installer YaCy. Les distributions GNU/Linux peuvent inclure, par
exemple un [paquet]{.underline} appelé
\[http://openjdk.java.net/install/ OpenJDK\]. Sinon, Java est disponible
sur le \[http://java.com/[fr]{.underline}/download Site web
[d\'Oracle]{.underline}\]. La version la plus récente est Java 8. La
version minimum de Java dont vous avez besoin pour YaCy est Java 7
(également connu sous le nom 1.7) - ce qui [peut être amené
à]{.underline} changer à l\'avenir.

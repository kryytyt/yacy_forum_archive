Der Name \"Bug\" für einen Programmierfehler hat eine \...
==========================================================

Date: 2016-05-19 20:28:03

Der Name \"Bug\" für einen Programmierfehler hat eine schöne Anekdote,
als [Grace Hopper tatsächlich eine Motte ein Relais gestört
hat](https://de.wikipedia.org/wiki/Programmfehler#.E2.80.9EBug.E2.80.9C_als_Synonym_f.C3.BCr_Programmfehler)
und daraufhin \"First actual case of bug being found\" notierte.

Und so passt es ganz schön, dass [jetzt eine Motte in der Wüste von
Nevada einen Tesla-Autopiloten außer Gefecht gesetzt
hat](http://www.autoblog.com/2016/05/12/tesla-driver-attacked-by-mothra-in-nevada-desert/).

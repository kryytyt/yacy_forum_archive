Podlove is an initiative to improve the overall technical infrastructure
for podcasting. Podlove is both an network for developers to discuss
features and agree on standards as well as an incubator for software and
file formats under the Podlove name. Tim Pritlove is founder and
coordinator of Project Podlove. While making just minor code
contributions he\'s behind the specifications, doing most of the public
communication, event organization and is behind most design documents
and feature requests.

Guest: [Tim Pritlove](http://podlove.org/)

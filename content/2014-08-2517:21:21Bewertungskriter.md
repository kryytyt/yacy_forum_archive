== These ==\
Der Crawler verhält sich was die Erfassung von Daten betrifft - wie ein
Hamster - alles wird gehortet. Eine Ausklammerung von bestimmten
Untermengen von Daten:\
\* auf der Dokumentenebene - bestimmte Bereiche eines Dokuments - bspw.
einer Webseite\
\* auf der Websiteebene - bestimmte Bereiche einer Website - bspw. nur
eine Sprache\
ist \_nicht\_ vorgesehen. Derartige großzügige Hortung von allen
verfügbaren Daten macht in einem Peer-to-Peer durchaus einen Sinn - in
einem Crawler-Durchgang werden alle verfügbare Daten erfasst - wer
weiss, Daten, die einem Peer im YaCy-Netz unwichtig sind, für anderen
Peer besonders interessant sind.\

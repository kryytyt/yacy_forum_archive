Oracle hat Java so monströs verkackt, dass die US-Regulierungsbehörde \...
==========================================================================

Date: 2015-12-22 19:27:54

Oracle hat Java so monströs verkackt, [dass die US-Regulierungsbehörde
FTC ihnen jetzt Vorschriften
macht](http://www.theregister.co.uk/2015/12/22/ftc_oracle_java/).
Vorschriften? Was denn für Vorschriften? Nun, [hier sind die
Details](https://www.ftc.gov/system/files/documents/cases/151221oracleorder.pdf).
Sie müssen einen peinlichen Brief an ihre Kunden auf ihrer Webseite
platzieren und sie müssen Schlangenölhersteller anschreiben, damit ihnen
die helfen, die Botschaft zu verbreiten.

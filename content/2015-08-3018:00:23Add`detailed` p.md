Add \`detailed\` parameter to /api/import.json search mode
==========================================================

Date: 2015-08-30 18:00:23

``` {style="white-space:pre-wrap;width:81ex"}
Add `detailed` parameter to /api/import.json search mode

When import api is called with `detailed=true`, the imported field will return all messages entries instead of the list of message ids in `imported` field.

Fix #120
```

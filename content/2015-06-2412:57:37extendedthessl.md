extended the /link - constraint: it is now possible to add a regular
====================================================================

Date: 2015-06-24 12:57:37

``` {style="white-space:pre-wrap;width:81ex"}
extended the /link - constraint: it is now possible to add a regular
expression in the form /link=regex. Example:
"freedom /link=.*fsfe.*"
returns only tweets with the word "freedom" containing a link which
point to an url containing the word "fsfe"
```

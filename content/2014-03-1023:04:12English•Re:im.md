The default admin name in YaCy is \"admin\". By default (if you have not
set an admin password) you should have access to your peer if YaCy and
your browser are running on the same machine. If this is the case and
you still can not log in, you can try to run the
[reconfigureYACY.sh]{style="font-weight: bold"} script (sorry, no
Windows version, but it should work ith Linux and OS X).

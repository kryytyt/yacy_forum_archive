===YaCy-Installer===

 

\[\[Bild:DownloadWindowsInstaller.png\|thumb\]\]

 

\[\[Bild:DownloadWindowsInstaller.png\|thumb\]\]

−

~~Mit dem YaCy~~-Installer kann YaCy auf dem unter Windows üblichen Weg mit einer .exe-Datei installiert werden. Bei der Installation wird überprüft, ob die benötigte Java-Laufzeitumgebung ~~breits~~ installiert ist. Ist dies nicht der Fall, wird diese ~~online~~ nachgeladen und installiert. Sollten keine Administrator-Rechte vorliegen, wird die Java-Installation in den Ordner \"Gemeinsame Dokumente\" (XP) bzw. \"Öffentliche Dokumente\" (Vista) abgelegt. Von dort kann dann ein Administrator Java auf dem System installieren.

+

[Der Windows]{.underline}-Installer [steht auf der \[http://yacy.net YaCy-Seite\] zum Download zur Verfügung (siehe Abbildung). Damit]{.underline} kann YaCy auf dem unter Windows üblichen Weg mit einer .exe-Datei installiert werden.[\
]{.underline}

 

+

Bei der Installation wird überprüft, ob die benötigte Java-Laufzeitumgebung [bereits]{.underline} installiert ist. Ist dies nicht der Fall, wird diese [automatisch]{.underline} nachgeladen und installiert. Sollten [für die Installation von YaCy]{.underline} keine Administrator-Rechte vorliegen, wird die Java-Installation in den Ordner \"Gemeinsame Dokumente\" (XP) bzw. \"Öffentliche Dokumente\" (Vista) abgelegt. Von dort kann dann ein Administrator Java auf dem System installieren[.\
]{.underline}

 

+

[YaCy wird standardmäßg in das Verzeichnis C:\\users\\\\\YaCy installiert. Auf dem Deskop werden ein YaCy-Icon und im Startmenü ein YaCy-Eintrag angelegt]{.underline}.

−

YaCy kann auch ~~manuell installiert werden. Hierzu steht ein komprimiertes Archiv (Dateiendung .tar.gz) zum Download bereit, welches alle benötigten Dateien enthält. Der Inhalt kann an beliebiger Stelle~~ mit ~~einem geeigneten Programm (z.B. FilZip, WinZip oder WinRAR) entpackt werden.~~

+

YaCy kann auch mit [dem generischen]{.underline} Paket, das für alle Betriebssysteme geeignet ist[, installiert werden]{.underline}. Es steht auf der \[http://yacy.net YaCy-Seite\] zum Download zur Verfügung (siehe Abbildung). [\
]{.underline}

−

 

+

[Zum Entpacken muss ein geeignetes Programm installiert sein (z.B. \[http://www.7-zip.org 7zip\]  FilZip, WinZip oder WinRAR). Das Archiv (Dateiendung .tar.gz) kann in ein geeignetes Verzeichnis unterhalb von C:\\users\\\ kopiert werden, um es dann mit Rechtsklick und \"hier entpacken\" zu entpacken. Ggf. wird es im ersten Schritt nur dekomprimiert und man erhält eine Archiv-Datei (Dateiendung .tar). Diese Archiv-Datei muss erneut entpackt werden, um das yacy-Verzeicnhis zu extrahieren.\
]{.underline}

−

~~Das Archiv ist ein generisches~~ Paket, das für alle Betriebssysteme geeignet ist. Es steht auf der \[http://yacy.net YaCy-Seite\] zum Download zur Verfügung (siehe Abbildung).\
\
\
\

+

[Beispiel: yacy\_v1.82.tar.gz (komprimiertes Archiv) =\> entpacken =\> yacy\_v1.82.tar (Archiv-Datei)=\> entpacken =\> yacy (Verzeichnis)]{.underline}

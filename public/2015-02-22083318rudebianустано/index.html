[== Настройка автоматического обновления ==]{.underline}

 

+

[При настройке этим способом, внутреннее авто обновление YaCy - не работает. Автоматическое обновление может быть выполнено средствами  ОС Ubnutu. например с помощью команды \[\[crontab\]\]. Для этого необходимо вставить следующую строку в файл: /etc/crontab .]{.underline}

 

+

[0 6 \* \* \* root apt-get update && apt-get -y \--force-yes install yacy]{.underline}

 

+

[]{.underline}

 

+

[В Ubuntu, в выше переведенной строке действительна только для системного файла crontab (/etc/crontab) - вы можете редактировать этот файл, на более новых ОС Ubuntu напрямую, без помощи команды crontab.]{.underline}

 

+

[]{.underline}

 

+

[In Ubuntu, the above line is only valid for the system crontab file (located at /etc/crontab) - you can edit  this file on newer Ubuntu OS directly without using the crontab command. ]{.underline}

 

+

[]{.underline}

 

+

[Ниже приведены комментарии для файла в Ubuntu 12.04 LTS:]{.underline}

 

+

[]{.underline}

 

+

[\# /etc/crontab: system-wide crontab]{.underline}

 

+

[]{.underline}

 

+

[\# В отличие от любой другой crontab Вы не должны запустить \`crontab\',]{.underline}

 

+

[\# чтобы установить новую версию, когда вы редактируете этот файл]{.underline}

 

+

[\# и файлы в /etc/cron.d. Эти файлы также имеют поля username,]{.underline}

 

+

[\# что ни один из других crontabs неделает.]{.underline}

 

+

[]{.underline}

 

+

[\# Unlike any other crontab you don\'t have to run the \`crontab\']{.underline}

 

+

[\# command to install the new version when you edit this file]{.underline}

 

+

[\# and files in /etc/cron.d. These files also have username fields,]{.underline}

 

+

[\# that none of the other crontabs do.]{.underline}

 

+

[]{.underline}

 

+

[В Ubuntu если вы хотите использовать \'\'\'crontab\'\'\' от пользователя \'\'\'root\'\'\', Выберете редактор которым вы будете редактировать crontab пользователя root:]{.underline}

 

+

[]{.underline}

 

+

[username\@hostname:\~\$ \'\'\'sudo crontab -e\'\'\']{.underline}

 

+

[no crontab for root - using an empty one]{.underline}

 

+

[]{.underline}

 

+

[Select an editor.  To change later, run \'select-editor\'.]{.underline}

 

+

[  1. /bin/ed]{.underline}

 

+

[  2. /bin/nano        \<\-\-\-- easiest]{.underline}

 

+

[  3. /usr/bin/vim.basic]{.underline}

 

+

[]{.underline}

 

+

 

+

[]{.underline}

 

+

[И добавьте следующую строку:]{.underline}

 

+

[]{.underline}

 

+

[0 6 \* \* \* /usr/bin/apt-get update && /usr/bin/apt-get -y \--force-yes install yacy]{.underline}

 

+

[]{.underline}

 

+

[Please note, there is no user name on the line above, and absolute (full) paths are used here to prevent binary location problems!]{.underline}

 

+

[]{.underline}

 

+

[Важно: проверьте что, нет ни одного имени пользователей в строках выше. Абсолютные (полные) пути используются здесь для того, чтобы предотвратить проблемы двоичный месте!]{.underline}

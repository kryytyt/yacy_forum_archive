The Emularity

Date: 2015-12-06 08:07:09

oldweb.today -- which, amazingly, is an actual URL that works -- lets you run ancient web browsers in your browser, but instead of pointing those old browsers at anachronistically modern versions of those pages, it pulls the correspondingly-old pages out of the Wayback Machine.

It runs the browsers by running a virtual Linux on the backend, that then runs emulators of the older operating systems inside it, that themselves run the browsers. It\’s turtles all the way down.

This is absolutely brilliant. Source here.

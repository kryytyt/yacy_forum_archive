Von mehreren Seiten kommt gerade die Verschwörungstheorie, ...

Date: 2013-06-09 20:53:31

Von mehreren Seiten kommt gerade die Verschwörungstheorie, dass Microsoft Skype möglicherweise auf Drängen der NSA gekauft habe, damit die endlich ordentlich abgeschnorchelt werden können. Das wird dann darauf zurückgeführt, dass Microsoft Skype im Mai 2011 gekauft hat, und laut der NSA-Folien Skype seit 2/6/11 an PRISM teilnimmt. Das sind aber Folien von Amerikanern. 2⁄6 heißt nicht der 2. Juni sondern der 6. Februar. Skype nimmt also laut dieser Folien schon länger an PRISM teil als sie in Besitz von Microsoft sind.

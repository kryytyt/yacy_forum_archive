=== Entpacken von YaCy ===

−

Nach dem \[http://www.yacy.net/~~yacy/Download.html~~ Herunterladen\] muss YaCy einfach nur an eine geeigneten Stelle entpackt werden. Programme zum Entpacken für Microsoft Windows sind z.B. \[http://www.filzip.com/ FilZip\], \[http://www.winzip.com/ WinZip\] oder \[http://www.winrar.de/ WinRAR\] und für Mac OS X \[http://www.stuffit.com/mac/index.html StuffIt Expander\]. Unter Linux kann man \[http://perso.wanadoo.fr/coquelle/karchiver/ kArchiver\] benutzen oder einfach die Befehle \'gunzip \.tar.gz\' und \'tar -xf \.tar\' (oder kurz \'tar -zxf \.tar.gz\') in der Kommandozeile eingeben.

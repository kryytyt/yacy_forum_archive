If you are using a VPS Cloud server it could be lack of swap space.

Try this.
cd /var
touch swap.img
chmod 600 swap.img
dd if=/dev/zero of=/var/swap.img bs=2048k count=1000
mkswap /var/swap.img
swapon /var/swap.img
echo \“/var/swap.img none swap sw 0 0\” >> /etc/ fstab

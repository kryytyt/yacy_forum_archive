Github scheint sich — zumindest bei Atom — auf ...

Date: 2015-08-12 00:44:44

Github scheint sich — zumindest bei Atom — auf einen anderen Code of Conduct festzulegen: diesen hier, Contributor Code of Conduct. Und der sieht mir auf den ersten Blick auch in Ordnung aus. Wobei ich mich ja immer noch frage, wozu wir überhaupt einen Code of Conduct brauchen. Nichts von den hier explizit gemachten Dingen ist in irgendeiner Art von zivilisierter Gesellschaft etwas anderes als völlig klar übergriffig und wird entsprechend geahndet. Wer sich wie ein Arschloch benimmt, fliegt raus. Ich sehe da nicht mal Anflüge einer Grauzone, die ein Explizitmachen irgendwie rechtfertigen würden. Es fällt allerdings auf, dass sie die Nationalität explizit nehmen, und damit gegen geltendee Gesetze verstoßen, denn die USA diskriminiert unilateral gegen Länder wie Syrien und den Iran und verbietet Technologietransfer in diese Länder. Eine Site wie Github kann die also, mal ganz pragmatisch betrachtet, gar nicht nicht diskriminieren. Und damit stellt sich dann die Frage, wieso wir ein solches Dokument überhaupt brauchen.

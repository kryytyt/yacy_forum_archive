[We want that our RPi starts YaCy at boot]{.underline} time [and shuts it down properly. Create a file in /etc/init.d/yacy with]{.underline} the [following content:]{.underline}

[case \"\$1\" in]{.underline}

[  start)]{.underline}

[    ;;]{.underline}

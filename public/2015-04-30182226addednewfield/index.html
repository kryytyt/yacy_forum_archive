Added new field definitions (for future use):
parent
assigns a id_str to refer to a parent message. This can be used to chain
messages to create a conversation in a tree-like shape
client
field can be used with the name of an user group; this can be used to
separate the message space into separate sub-groups
texts
a field to list text attachments
audio
a field to list audio attachments
videos
a field to list video attachments
campaign
a field to list campaign attachments

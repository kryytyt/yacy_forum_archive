Microsoft sitzt anscheinend seit über nem halben Jahr ...

Date: 2014-05-21 23:32:14

Microsoft sitzt anscheinend seit über nem halben Jahr auf einer IE Remote Code Execution Lücke. Unentschuldbar. Besonders in Zeiten, in denen Chrome innerhalb von Stunden und Firefox innerhalb von Tagen fixt. Gut, auf die Versionen mit Fix muss man dann ne Woche (Chrome) oder ein-zwei Monate (Firefox) warten. Immer noch kein Vergleich dazu, was Microsoft hier für eine Schnarchnasigkeit an den Tag legt.

Ich an Microsofts Stelle hätte da sofort gefixt, denn so sieht es nach \“die NSA hat um noch ein paar Wochen Zeit gebeten\” aus.

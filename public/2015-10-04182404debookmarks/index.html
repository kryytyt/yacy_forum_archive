Um den Index anzureichern, können Crawls mit geeigneten Startpunkten gestartet werden. Dazu eignen sich eigene Bookmarks gut, die man in YaCy importieren kann.\ \ ==Bookmarks beziehen==\ Im ersten Schritt müssen Bookmarks aus dem Browser exportiert werden.\ Im Firefox ruft man mit der Tastenkombination „Strg + Umschalt + o" die Lesezeichenbibliothek auf. Hier wählt man den Menüpunkt „Importieren und sichern aus, und hier den Untermenüpunkt „Lesezeichen nach HTML exportieren". Nach Auswahl des Speicherorts wird die Lesezeichenliste als html-Datei gespeichert.\ \ ==Bookmarks importieren==\ \[\[Bild:Bookmarks importieren.jpg\]\]\ Im zweiten Schritt wird nun diese Liste in YaCy importiert. Wählen Sie zunächst in der Navigationsleiste unter „ Production" den Eintrag „Advanced Crawler" aus. Es wird die Seite „Expert Crawl Start" angezeigt, in der Sie einen Crawl-Job anlegen können. Aktivieren Sie unter „Start Point" die Option „From File (enter a path within youtr local file system)" und geben Sie im Eingabefeld daneben den Pfad zu der html-Datei mit den Bookmarks an. Passen Sie ggf. weitere Parameter für den Expert Crawl an und klicken Sie ganz unten auf die Schaltfläche „Start New Crawl Job".\ Sie können den Crawler Monitor in der Navigationsleiste unter „Monitoring" über den Eintrag „Crawler Monitor" aufrufen, um den gestarteten Crawl zu beobachten oder ggf. zu beenden.a
